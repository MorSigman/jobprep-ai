# JobPrep AI

מערכת RTL עברית לניהול תהליך חיפוש עבודה אישי — מעקב אחר משרות, ארגון מידע לפי משרה, והכנה מובנית לשלבי המיון.

---

## הצורך

בחיפוש עבודה, המידע מתפזר בין לוחות משרות, גרסאות קורות חיים, הערות, תזכורות מעקב, תיאורי תפקיד ומשימות הכנה. אין מקום אחד שמאגד את כל המידע לפי משרה, בעברית, בפריסת RTL.

JobPrep AI נועד לארגן את התהליך הזה במקום אחד.

---

## גרסת הדמו הנוכחית

הגרסה הנוכחית ב-GitHub היא **גרסת דמו בלבד** — כל הנתונים המוצגים בה פיקטיביים. אין נתונים אישיים אמיתיים בשום מקום בקוד.

### תכונות קיימות

- לוח בקרה בעברית RTL
- 5 כרטיסי סטטיסטיקה חיים (סה"כ / היום / השבוע / בתהליך ראיון / ממתין)
- גרף בקשות לפי יום (CSS בלבד, ללא ספריית גרפים)
- גרף עוגה לפי קטגוריית משרה (CSS conic-gradient)
- עמוד מעקב משרות עם חיפוש וסינון לפי סטטוס
- עמוד פרטי משרה עם לשוניות הכנה לשיחה טלפונית, ראיון טכני וראיון אישי
- עמוד פרויקטים עם דוגמאות דמו ונקודות הכנה
- עמוד ראיון אישי עם כרטיסי תשובות מובנים
- פאנל AI לדמו — ממשק ויזואלי בלבד, ללא קריאת API אמיתית
- מצב בהיר / כהה (CSS variables)
- ניווט מקלדת נגיש

## כיוון עתידי — Local-First

הגרסה האמיתית של האפליקציה תרוץ **מקומית על המחשב של המשתמש**.

**תוכנית אחסון עתידית:**

| שלב | פתרון | מה זה נותן |
|---|---|---|
| אב-טיפוס ראשון | `localStorage` | נתונים שורדים רענון עמוד |
| אחסון מקומי מלא | IndexedDB | אין מגבלת גודל, מבנה נתונים מסוג, ללא שרת |
| גיבוי | ייצוא / ייבוא JSON | המשתמש שולט בגיבוי שלו |
| גרסת Desktop (עתידי) | Electron / Tauri + SQLite | אפליקציה מקומית מלאה |

אין עתיד לאחסון בענן, אין חשבון משתמש, אין שרת חיצוני.

## ארכיטקטורה

**גרסה נוכחית:**
```
דפדפן
  └── React UI
       └── נתוני דמו (קבועי TypeScript ב-src/data/)
```

**גרסה עתידית:**
```
דפדפן
  └── React UI
       ├── IndexedDB (נתוני משתמש — מקומי בלבד)
       └── ייצוא / ייבוא JSON (גיבוי בשליטת המשתמש)
```

ניווט בין עמודים מבוסס `useState` (ללא React Router בשלב זה). הפריסה היא RTL מלא — `dir="rtl"` על מעטפת האפליקציה.

## טכנולוגיות

| שכבה | בחירה |
|---|---|
| ממשק | React 19 + TypeScript |
| בנייה | Vite |
| עיצוב | CSS עם custom properties (ללא ספריית CSS) |
| גרפים | CSS בלבד — `conic-gradient`, flexbox |
| ניווט | `useState` state machine |
| נתונים (דמו) | קבועי TypeScript מקומיים |
| נתונים (עתיד) | IndexedDB |
| פריסה | RTL מלא — עברית |

## הרצה מקומית

```bash
# התקנה
npm install

# שרת פיתוח
npm run dev

# בנייה לייצור
npm run build
```

פתח את [http://localhost:5173](http://localhost:5173). כל הנתונים המוצגים הם נתוני דמו פיקטיביים המוגדרים ב-`src/data/demoJobs.ts`.
